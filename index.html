    COVID-19 Diagnosis Analysis using Transfer Learning

    Automatic Diagnosis System implemented to control the number of daily cases of COVID-19. There are only a limited number of test kits available in hospitals. Where, this application is played vital role to identify cases and control to spread further. During this study, I have worked on three pre-trained neural networks supported convolutional neural networks (VGG16, VGG19, ResNet50) are proposed for detecting Coronavirus pneumonia infected patients through X-rays and computerized tomography (CT). By using cross-validation, we've got implemented binary classifications with two classes (COVID-19, Normal (healthy)). Taking into consideration the results obtained, the pre-trained ResNet50 model provides the classification performance (97.77% accuracy, 100% sensitivity, 93.33% specificity, 98.00% F1- score) among the opposite three used models over 6259 images.
